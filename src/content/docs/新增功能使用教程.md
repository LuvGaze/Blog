## 1. 网站导航

### 新增网站条目

在 `src/content/daohang/` 目录下创建 `.md` 文件：

```markdown
---
name: 网站名称
url: https://example.com
description: 网站描述
icon: "fa7-brands:github"
category: 常用网站
order: 1
---
```

### 可用分类

- `我的网站`
- `常用网站`
- `资源网站`
- `工具网站`
- `大佬博客`
- `学习网站`
- `文档`
- `其他网站`

### 可用图标

- 使用 `fa7-brands:*` 格式的 FontAwesome 图标
- 使用 `material-symbols:*` 格式的 Material Symbols 图标
- 或使用外部图片 URL

## 2. 笔记本

### 创建新笔记本

1. 在 `src/content/life/notebooks/` 目录下创建文件夹，例如 `my-notebook/`

2. 在文件夹内创建 `_index.json` 文件：

```json
{
  "name": "我的日记本",
  "summary": "记录生活点滴",
  "cover": "cover.jpg"
}
```

3. 在文件夹内创建 `.md` 文件作为日记条目：

```markdown
---
name: 日记标题
date: 2024-01-15
tags: ["生活", "感悟"]
image: ["photo1.jpg", "photo2.jpg"]
---

日记内容...
```

## 3. 书架

### 新增书籍

在 `src/content/bangumi/books/` 目录下创建 `.md` 文件：

```markdown
---
title: "书籍名称"
category: "book"
image: "https://example.com/cover.jpg"
score: 9.5
status: 2
comment: "我的评价"
tags: ["科幻", "小说"]
published: "2024-01-01"
---

书籍简介...
```

### status 状态值

- `1` - 想看（wish）
- `2` - 看过（collect）
- `3` - 在看（doing）
- `4` - 搁置（on_hold）
- `5` - 抛弃（dropped）

## 4. 影视与游戏

### 新增影视作品

在 `src/content/bangumi/movies/` 目录下创建 `.md` 文件：

```markdown
---
title: "电影名称"
category: "real"
subcategory: "movie"
image: "https://example.com/cover.jpg"
score: 9.4
status: 2
comment: "我的评价"
tags: ["科幻", "冒险"]
published: "2014-01-01"
---

电影简介...
```

### 新增游戏

在 `src/content/bangumi/games/` 目录下创建 `.md` 文件：

```markdown
---
title: "游戏名称"
category: "game"
image: "https://example.com/cover.jpg"
score: 9.8
status: 3
comment: "我的评价"
tags: ["冒险", "开放世界"]
published: "2024-05-12"
---

游戏简介...
```

### category 分类值

- `book` - 书籍
- `anime` - 动漫
- `game` - 游戏
- `real` - 影视

### subcategory 子分类（仅影视）

- `movie` - 电影
- `tv` - 电视剧
- `anime` - 动漫
- `documentary` - 纪录片

## 5. 更新日志

### 新增更新记录

在 `src/content/changelog/` 目录下创建 `.md` 文件：

```markdown
---
version: "v1.0.1"
date: 2024-01-15
time: "10:30"
type: feature
description: "新增功能描述"
---
```

### type 类型值

- `feature` - 新功能
- `improvement` - 改进
- `fix` - 修复
- `removal` - 删除

## 8. 音乐播放器

### 配置歌单

修改 `src/config/musicConfig.ts` 文件：

```typescript
playlists: [
  { 
    id: "daily", 
    name: "每日推荐", 
    server: "netease", 
    type: "playlist", 
    playlistId: "8659142201" 
  },
  { 
    id: "classical", 
    name: "古典音乐", 
    server: "tencent", 
    type: "playlist", 
    playlistId: "5575587639" 
  },
]
```

### 支持的平台

- `netease` - 网易云音乐
- `tencent` - QQ音乐
- `kugou` - 酷狗音乐

## 注意事项

1. **图片路径**：使用外部 URL 或相对于 `src/content/` 目录的相对路径
2. **日期格式**：使用 `YYYY-MM-DD` 格式
3. **排序**：大多数功能按日期倒序排列，最新的在前面
4. **缓存**：修改内容后需要重启开发服务器或重新构建
5. **图标**：优先使用项目已有的图标库，避免外部依赖
