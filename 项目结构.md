## 二、配置文件说明

### 2.1 配置文件概览

| 配置文件 | 说明 | 主要功能 |
| :--- | :--- | :--- |
| `siteConfig.ts` | 站点基础配置 | 站点标题、描述、页面开关、社交链接 |
| `navBarConfig.ts` | 导航栏配置 | 导航链接、菜单结构、LinkPresets |
| `sidebarConfig.ts` | 侧边栏配置 | 侧边栏模块、友链、标签云 |
| `profileConfig.ts` | 个人资料配置 | 头像、简介、社交账号 |
| `backgroundWallpaper.ts` | 背景壁纸配置 | 壁纸图片、切换间隔 |
| `commentConfig.ts` | 评论系统配置 | 评论服务提供商、配置参数 |
| `musicConfig.ts` | 音乐播放器配置 | 歌单、播放模式、导航栏显示 |
| `fontConfig.ts` | 字体配置 | 页面字体、代码字体 |
| `coverImageConfig.ts` | 封面图片配置 | 文章封面、默认封面 |
| `expressiveCodeConfig.ts` | 代码块配置 | 代码高亮、主题 |
| `effectsConfig.ts` | 特效配置 | 页面特效、动画 |
| `announcementConfig.ts` | 公告配置 | 顶部公告条内容 |
| `footerConfig.ts` | 页脚配置 | 页脚内容、版权信息 |
| `licenseConfig.ts` | 许可证配置 | 内容许可证、署名方式 |
| `friendsConfig.ts` | 友链配置 | 友情链接列表 |
| `sponsorConfig.ts` | 打赏配置 | 打赏方式、二维码 |
| `analyticsConfig.ts` | 统计分析配置 | 流量统计、访问分析 |
| `galleryConfig.ts` | 相册配置 | 相册分类、图片路径 |
| `plantumlConfig.ts` | PlantUML配置 | 图表渲染、主题 |
| `pioConfig.ts` | Live2D配置 | 看板娘模型、交互 |
| `externalBangumiConfig.ts` | 外部番组配置 | Bilibili、TMDB集成 |

```typescript
      id: "daily",
      name: "每日推荐",
      server: "netease",  // netease | tencent | kugou | xiami | baidu

      playlistId: "8659142201",
      cover?: "https://...",  // 歌单封面（可选）
    },
  ],
  
  meting: {              // Meting API配置
    api: "https://api.i-meto.com/meting/api?...",
    server: "netease",
    type: "playlist",
    id: "8659142201",
  },
}
```

**导航栏音乐按钮开关**：设置 `showInNavbar: false` 即可隐藏导航栏右侧的音乐按钮。

## 五、二次开发指南

### 5.1 启用/禁用功能

**步骤**：
1. 打开 `src/config/siteConfig.ts`
2. 在 `pages` 对象中找到对应的开关
3. 设置为 `true`（启用）或 `false`（禁用）

**示例**：
```typescript
pages: {
  books: true,    // 启用书架
  games: false,   // 禁用游戏
}
```

### 5.2 添加新页面

**步骤**：
1. 在 `src/pages/` 下创建新页面文件（如 `new-page/index.astro`）
2. 在 `src/config/siteConfig.ts` 中添加页面开关
3. 在 `src/config/types/siteConfig.ts` 中添加类型定义
4. 在 `src/config/navBarConfig.ts` 中添加导航链接
5. 在 `src/i18n/i18nKey.ts` 中添加翻译键
6. 在 `src/i18n/languages/` 各语言文件中添加翻译

### 5.4 修改导航栏

**添加导航链接**：
1. 在 `src/config/navBarConfig.ts` 的 `LinkPresets` 中添加新链接
2. 在 `links` 数组中添加到对应菜单位置

**示例**：

```typescript
links.push({
  name: "记录",
  url: "#",
  icon: "material-symbols:camera-outdoor",
  children: [
    ...(siteConfig.pages.books ? [LinkPresets.Books] : []),
    ...(siteConfig.pages.newPage ? [LinkPresets.NewPage] : []),
  ].filter(Boolean),
});
```

3. 在组件中使用 `i18n(I18nKey.yourKey)` 获取翻译

## 七、数据统计与排序

### 7.1 规划排序
- 使用 `order` 字段进行排序
- 数字越小，排序越靠前

### 7.2 更新日志排序
- 按 `date` 字段降序排序（新日期在前）
- 相同日期按 `time` 字段降序排序

### 7.3 书籍/影视/游戏排序
- 按 `date` 字段降序排序
- 相同日期按 `time` 字段降序排序

### 7.4 足迹统计
- 地图颜色深浅根据省份 `visitCount` 总和计算
- 列表显示到访次数和日期
